
# 🚀 Modern Minimalist Portfolio

En högpresterande, responsiv portfolio byggd med fokus på estetik, mjuka animationer och en "dark mode first"-upplevelse. Designad med en ren estetik som lyfter fram projekt och teknisk kompetens.

## ✨ Funktioner

* **Lenis Smooth Scroll:** En silkeslen scroll-upplevelse för modern webbkänsla.
* **Interaktiva Partiklar:** En dynamisk bakgrund med `canvas`-partiklar som reagerar på rörelse.
* **Responsiv Design:** Fullt optimerad för allt från 4K-skärmar till mobila enheter.
* **Modern Tech Stack UI:** Stilrena kort för att visa upp tekniska färdigheter.
* **Glassmorphism & Gradients:** Subtila badges och gradients som ger djup åt designen.

## 🛠 Tech Stack

* **Frontend:** HTML5, CSS3 (Custom Variables), JavaScript (ES6+)
* **Bibliotek:** [Lenis](https://github.com/darkroomengineering/lenis) för scroll, [GSAP](https://greensock.com/gsap/) (valfritt för animationer)
* **Typsnitt:** Plus Jakarta Sans
